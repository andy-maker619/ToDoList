# To Do List
Este proyecto es una aplicacion para android de un gestor de tareas, utilizando una base de datos en sqlite.
# Instalacion y ejecucion
1. Tener instalado git
2. Copiar el link del repositorio desde github
3. Abrir el Android Studio
4. En la pesta√±a de File, seleccionaremos la opcion de New y luego en Proyect from Control Version
5. Luego solo le pegamos el link de github y se clonara.
6. Una vez ya en el proyecto en Android Studio tendra que sincronizar el proyecto con los archivos de gradle en el apartado de File y luego en la opcion de Sync proyect with gradle files
7. Ahora podra ejecutar la app sin problemas.
# Base de datos
Este proyecto nomas maneja una tabla llamada tasks.
| task |
|----------|
| id    |
| task_title    |
| task_description    |
| created_at    |
| is_complete    |
